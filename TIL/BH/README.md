### 배운 것
- 최적 길찾기 알고리즘 (A*) : 다익스트라 알고리즘을 발전시킨 알고리즘
    - 출발 노드와 도착 노드 사이의 비용을 계산
    - Heuristic cost function : f(n) = g(n) + h(n)
    - g(n) = 유클리디안 거리 / h(n) = 맨하탄 거리
    - 추가 공부 필요
